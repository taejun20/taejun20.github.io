# taejun20.github.io

Personal academic website for Taejun Kim. Live at [taejunkim.com](https://taejunkim.com).

## Stack

Pure static HTML/CSS/JS — no build tools required. Dependencies loaded from CDN:

- [Bootstrap 5](https://getbootstrap.com/)
- [Bootstrap Icons](https://icons.getbootstrap.com/)
- [marked.js](https://marked.js.org/) (Markdown rendering)
